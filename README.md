# Eshoppers

Full Stack ECommerce Application with Stripe & Sanity
![eCommerce](https://user-images.githubusercontent.com/70088342/160780701-7bb38a57-76bd-49a2-a4ec-49f89c50a7c7.png)


This is a fully responsive ECommerce website with modern design and animations , with payment  and all advanced card functionalities feature ,completely integrated with Stripe to cover real payments. It also features adding and edit products on the go using a CMS.

- Technologies used are: Javascript, CSS, Shell
- Sanity is the unified content platform that’ll make the making of our entire app possible. <show sanity desk>
- Sanity allows us to focus on developing the application without having to worry about the content, file storage, and databases. They’ll cover the dirty work for us and allow us to build scalable and modern e-commerce web applications extremely easily.
